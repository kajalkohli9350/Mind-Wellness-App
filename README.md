
A frontend-based Mental Wellness Web Application developed using React JS to promote emotional well-being, stress reduction, and mental health awareness through interactive and user-friendly features.

Mental health is an important part of overall well-being. This application helps users manage stress and emotions by providing tools such as daily motivation quotes, mood tracking, breathing exercises, relaxing music, and dark/light mode support.

The project focuses on UI/UX design and frontend development concepts.
